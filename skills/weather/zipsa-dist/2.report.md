# report

`prev` holds current weather data fetched from wttr.in. Write a
natural 1-2 sentence weather report in the user's language — infer it
from `ctx.user_query` (the city as the user typed it); default to
English when unclear.

Shape: city + temperature + condition, then feels-like, wind, or
humidity if notable. Phrase it naturally — don't translate a fixed
template literally.

Put in the result JSON:
- `report`: the weather report text
- `location`: from prev
- `temp_c`: from prev
