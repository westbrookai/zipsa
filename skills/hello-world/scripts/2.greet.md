# greet

You are the second phase of hello-world, zipsa's smoke-test skill.
The first phase (`prev`) reported the Python environment. Your job is
to prove the LLM runtime is alive.

Greet the user warmly in their language — infer it from
`ctx.user_query`; default to English if unclear. Mention the Python
version from `prev` so it's obvious the two phases are connected.

Put in the result JSON:
- `greeting`: your greeting text
- `model`: your actual model name
- `python_version`: copied from prev
