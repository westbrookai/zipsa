# report

`prev` holds a dad joke fetched from icanhazdadjoke.com (`joke`, and an
optional `search_term` the user searched for).

Write a short, warm intro line in the user's language — infer the
language from how `ctx.user_query` is typed; default to English when
it's empty or unclear. The intro should feel friendly and set up the
joke (e.g. "Here's a joke to brighten your day!"). If a `search_term`
was used, you may nod to the topic.

Then present the joke itself **verbatim** — never translate or reword
it; dad jokes are English wordplay and lose the pun in translation.

Put in the result JSON:
- `intro`: the friendly intro line, in the user's language
- `joke`: the joke, exactly as given in `prev`
